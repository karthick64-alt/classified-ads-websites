# Classified Ads Website

A modern, feature-rich classified ads website with unique designs for each page.

## Features

- User Registration & Login with Social Login
- Post and Manage Ads
- Category & Subcategory System
- Advanced Filtering & Search
- Real-time Messaging
- Admin Panel
- Payment & Subscription System
- Responsive Design with TailwindCSS

## Getting Started

1. Install dependencies (optional for live-server):
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Open `index.html` in your browser or use the live-server.
